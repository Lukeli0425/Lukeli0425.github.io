Video-Audio Signal Processing
[Code]

This is my course project of Introduction to Auditory-visual Information System. I built three algorithms that solves the the following problems respectively:
1. Recognize faces from videos clips.
2. Recongizes voices from audios.
3. Given a video of three speakers speaking at the same time, separate the speeches from the three speakers.
This project is implemented in Python with the aid of Face Recognition, Resemblyzer and Speechbrain.
Astro Image Processing
[Code & Data]

This repo contains my code and data for astronomical image reduction. The images were taken at Huairou, Beijing with a 25cm telescope of DoA. Targets includes M13, M51, M3, etc.
Video Editing Based on Rhythm Matching
[Code] [Video]

This is my 2021 Signals and Systems course project. The main task is to build an algorithm that creates an video from a given set of video clips and a background music so that the video clips matches the background music in terms of audio rhythm as best as possible. This project is built with Matlab.
Stock Forecast with GPR
[Code]

This is my course project of Stochastic Processes. I built an algorithm using Guassian Process Regression that preditcs the stock prices based on previous observations. This project is implemented with Python and Scikit Learn.
